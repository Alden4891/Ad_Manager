Function Reference

_AD_UnlockObject

Unlocks an AD object (user account, computer account).

#Include <AD.au3>
_AD_UnlockObject($sObject)

 

Parameters

$sObject User account or computer account to unlock (sAMAccountName or FQDN)

 

Return Value

Success: 1
Failure: 0, sets @error to:
    1 - $sObject does not exist
    x - Error returned by SetInfo method (Missing permission etc.)

 

Remarks

None.

 

Related

_AD_IsObjectLocked, _AD_GetObjectsLocked
