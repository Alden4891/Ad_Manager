Function Reference

_AD_SetPasswordExpire

Sets user's password as expired or not expired.

#Include <AD.au3>
_AD_SetPasswordExpire($sUser[, $iFlag = 0])

 

Parameters

$sUser User account (sAMAccountName or FQDN)
$iFlag [optional] Sets the user's password as expired ($iFlag = 0) or as not expired ($iFlag = -1) (Default = 0)

 

Return Value

Success: 1
Failure: 0, sets @error to:
    1 - $sUser does not exist
    2 - $iFlag has an invalid value
    x - Error returned by SetInfo method (Missing permission etc.)

 

Remarks

When the user's password is set to expired he has to change the password at next logon
