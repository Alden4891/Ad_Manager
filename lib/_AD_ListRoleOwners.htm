Function Reference

_AD_ListRoleOwners

Returns a one-based array of FSMO (Flexible Single Master Operation) Role Owners.

#Include <AD.au3>
_AD_ListRoleOwners()

 

Parameters

None.

 

Return Value

Success: Returns an one-based one dimensional array of FSMO Role Owners. The array contains:
    1 - Domain PDC FSMO
    2 - Domain Rid FSMO
    3 - Domain Infrastructure FSMO
    4 - Forest-wide Schema FSMO
    5 - Forest-wide Domain naming FSMO

 

Remarks

None.
