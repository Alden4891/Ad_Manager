Function Reference

_AD_AddUserToGroup

Adds a user, group or computer to the specified group or computer.

#Include <AD.au3>
_AD_AddUserToGroup($sTarget, $sObject)

 

Parameters

$sTarget Group or computer (FQDN or sAMAccountName) where $sObject should be added to
$sObject User, group or computer to be added to $sTarget (FQDN or sAMAccountName)

 

Return Value

Success: 1
Failure: 0, sets @error to:
    1 - $sTarget does not exist
    2 - $sObject does not exist
    3 - $sObject is already a member of $sTarget
    x - Error returned by Add or SetInfo method (Missing permission etc.)

 

Remarks

The sAMAccountname of a computer requires a trailing "$" before converting it to a FQDN.
