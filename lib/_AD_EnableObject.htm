Function Reference

_AD_EnableObject

Enables an AD object (user account, computer account).

#Include <AD.au3>
_AD_EnableObject($sObject)

 

Parameters

$sObject User account or computer account to enable (sAMAccountName or FQDN)

 

Return Value

Success: 1
Failure: 0, sets @error to:
    1 - $sObject does not exist
    x - Error returned by SetInfo method (Missing permission etc.)

 

Remarks

None.

 

Related

_AD_IsObjectDisabled, _AD_DisableObject, _AD_GetObjectsDisabled
