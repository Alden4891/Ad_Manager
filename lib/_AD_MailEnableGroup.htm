Function Reference

_AD_MailEnableGroup

Mail-enables a Group.

#Include <AD.au3>
_AD_MailEnableGroup($sGroup)

 

Parameters

$sGroup Groupname (sAMAccountName or FQDN)

 

Return Value

Success: 1
Failure: 0, sets @error to:
    1 - $sGroup does not exist
    x - Error returned by SetInfo method (Missing permission etc.)

 

Remarks

None.
